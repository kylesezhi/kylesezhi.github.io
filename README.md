# Usage

## Update
1. If you need a new ruby, do: `rvm install [version number]`
2. `xcode-select install`
2. `bundle update`


## Serve locally
- `bundle exec jekyll serve`

## Content locations

### About

Images are in `/img/about/`

### Team

Team members and info are in `_config.yml`

Images are in `/img/team/`
